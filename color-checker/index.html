      Contrast grid

      176px

      Swatches and samples are recolored using published approximation matrices for each color-vision deficiency (a luma-based grayscale for achromatopsia). This is a visual aid, not a clinically accurate simulation.

      The simulated ratio re-runs the WCAG contrast formula on those recolored values. WCAG's contrast math only looks at luminance, so a pair can pass the real ratio yet still be hard to tell apart under a color vision deficiency that collapses their hues — the simulated ratio surfaces that gap. Treat it as a sanity check, not a certification.

      AA normal text ≥ 4.5:1 AAA normal text ≥ 7:1 AA L large/bold text ≥ 3:1 AAA L large/bold text ≥ 4.5:1

      Rows are used as the background; columns are used as the foreground (text).
